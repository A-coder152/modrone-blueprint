just a scope md file to define project goals and steps and stuff

### project goal
modular drone structure that can fly (shocker)
- fc should be able to handle battery power, motors, and position/altitude data
- power structure should be simple (rechargable batteries, and adding more should be relatively easy if needed)
- drone can be assembled/dissassembled quickly (and nice for packaging)
- drone can have different structural shapes (ofc disassembly is needed but the process for a new strucutre should be super easy compared to building a new drone from scratch)
- should have a processor for... purposes outside the scope of this submission (current plan is a raspi zero 2 w)

### blueprint goal
have the drone architecture designed, with all parts/blocks/connectors done on a sample drone shape.
basically the sample drone should be "construction ready"

### steps
1. standardize the drone connections
2. drone fc, building off of the guided project
3. fc/control block
4. other blocks (aux, ai, motors) and connectors
5. sample drone
6. code ofc
